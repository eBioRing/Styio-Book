# Aggregation

| Keyword | Aggregation                    |
| ------- | ------------------------------ |
| sum     | Compute the sum                |
| count   | Count the size of a collection |
| avg     | Compute the average            |
| max     | Keep the maximum value         |
| min     | Keep the minimum value         |
| first   | Keep the first value           |
| last    | Keep the last value            |
